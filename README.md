# slk

Command-line power tool for [slack](https://slack.com).

## Building and Installing

```shell
npm install -g slk
```
##  Usage: slk [options] [command]

### Commands:

*listen <channel>* listen to a channel
*post <channel> <username> <message>* post to a channel

### Options:

*-h, --help*           output usage information
*-t, --token <token>*  set the authentication token
